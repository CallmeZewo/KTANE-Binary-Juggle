On the Subject of Binary Juggle

This module takes "juggling numbers" to another level.

The module will start juggling an even number of balls. Each ball has 4 informations you need to note down in order to solve the module.

  • Color
  • Symbol
  • Rotation
  • Direction

With these 4 characteristics of each ball, determine the corresponding binary digit from the table below.

Clockwise Counterclockwise
Left Red 1 0 1 1 0 1 0 0
Green 0 1 1 0 1 0 0 1
Blue 1 1 0 0 0 1 1 0
Magenta 0 0 1 1 1 0 1 1
Right Red 1 0 0 1 0 0 1 1
Green 0 1 0 0 1 1 0 0
Blue 1 0 1 0 1 1 0 1
Magenta 0 1 0 1 0 0 1 0

Determine the amount of juggling

  • If the amount of balls moving to the right is greater than balls that are rotating counterclockwise, juggle according to how many 1s are in your binary string.
  • Otherwise, if the amount of balls moving to the left is greater than balls that are rotating clockwise, juggle based on the number of 0s in your binary string.
  • Otherwise, juggle based on the difference between the number of 1s and 0s.

How to Juggle numbers the correct way

  1. To juggle the binary string, split the string in the middle into two equal halves.
0110101011
→
01101 | 01011
  1. For each juggle, take the leftmost digit of the right half and append it to the left side of the left half.
01101 | 01011
→
001101 | 1011
  1. Then take the rightmost digit of the left half and append it to the right side of the right half.
001101 | 1011
→
00110 | 10111

Repeat steps 2 and 3 according to your determined juggle amount. After you finish juggling, join the two halves back together and press every ball whose position corresponds to a 1 in the final binary string. When all the correct balls are pressed, the module is solved.
